EISA ANWAR

Wheelchair Activity Tracking System

TO RUN PROGRAM
	1. Download the source code
	2. Open in Arduino IDE
	3. Download libraries for all components  from their official pages
		- with the exception of the optical rotary encoders (library location: https://github.com/PaulStoffregen/Encoder
	4. Upload Program To Arduino DUE Microcontroller ensuring  all components  are attached correctly
